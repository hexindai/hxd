# hxd

Mysql数据字典导出和http日志文件查看工具

```
➜ hxd -h

1.导出数据字典的功能，当前只支持mysql
2.动态实时查看基于http协议的日志文件

Usage:
  hxd [command]

Available Commands:
  dump        导出数据库的数据字典并生成excel文件
  help        Help about any command
  tail        动态查看日志文件

Flags:
  -h, --help   help for hxd

Use "hxd [command] --help" for more information about a command.
```
